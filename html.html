HTML Tutorial

1. What is HTML?

HTML (HyperText Markup Language) is the skeleton of any webpage. It defines the structure, layout, and content of a site.

2. Basic Structure of HTML

Every webpage has this structure:


<html>
    <head>
        <title>My First Page</title>
    </head>
    <body>
        <h1>Hello World!</h1>
    </body>
</html>

3. What are HTML Tags?

Tags tell the browser what to display.

  • <h1> → Heading
  • <p> → Paragraph
  • <img> → Image
  • <a> → Link

4. Example Program

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>
<a href="https://google.com">Click Me</a>
