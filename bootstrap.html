1. What is Bootstrap?

Bootstrap is a CSS framework used to make websites responsive and beautiful without writing much CSS.

2. Bootstrap Grid System

Bootstrap divides a page into 12 columns. You can create rows and columns for layout. 

<div class="row">
    <div class="col-6">Left</div>
    <div class="col-6">Right</div>
</div>
