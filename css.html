CSS Tutorial — Topic Wise

1. What is CSS?

CSS (Cascading Style Sheets) is used to style your web pages — colours, fonts, spacing, borders, animations, layouts etc.

2. Types of CSS

  • Inline CSS — inside the tag
  • Internal CSS — inside <style>
  • External CSS — in a .css file

3. Common Properties

  • color: changes text colour
  • background-color: sets background
  • font-size: controls size
  • padding: inner spacing
  • margin: outer spacing

4. Example

h1 {
    color: red;
    background: yellow;
    padding: 20px;
    text-align: center;
}
